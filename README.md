# Alt::Params::Util::PLICEASE ![static](https://github.com/uperl/Alt-Params-Util-PLICEASE/workflows/static/badge.svg) ![linux](https://github.com/uperl/Alt-Params-Util-PLICEASE/workflows/linux/badge.svg)

Simple, compact and correct param-checking functions

# SYNOPSIS

env PERL\_ALT\_INSTALL=OVERWRITE cpanm Alt::Params::Util::PLICEASE

# DESCRIPTION

This module provides an alternate implementation for [Params::Util](https://metacpan.org/pod/Params::Util).
It will only be installed if you explicitly set `PERL_ALT_INSTALL`
to `OVERWRITE`.  [Params::SomeUtil](https://metacpan.org/pod/Params::SomeUtil) has details on how the two
implementations differ.
